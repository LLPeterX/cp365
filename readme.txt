Назначение программы:
 - Обработка входящих файлов из ФНС
 - Формирование исходящих файлов в ФНС

Обработка входящих файлов (arj):
 - распаковать входящие файлы AFN_MIFNS000_...ARJ
 - расшифровать
 - снять ЭЦП
 - поместить расшифрованные файлы *.xml во входящий каталог АБС
 - при необходимости сформировать PB1

Обработка входящих файлов (ПТК ПСД):
 - загрузить список ИЭС2 посылок mz из базы данных ПТК ПСД
 - снять ЭЦП ПДТ ПСД
 - выгрузить файлы AFN_MIFNS00..ARJ в каталог входящих файлов
 - выполнить пункты по расшифровке и т.п. - см. выше

Обработка исходящих файлов:
 - постановка ЭЦП на исходящих файлах через СКАД "Сигнатура"
 - шифрование
 - формирование архивных файлов
 - подпись исходящих файлов

Особенности:
 - использование для криптографии эмулятора FDD imdisk
 - использование ПТК ПСД

Зависимости:
 - СКАД "Сигнатура" 5.x
 - Microsoft .Net Framework 4.7
 - (не обязательно) установленый эмулятор FDD imdisk
 - (не обязательно) rawwritewin (для снятия образов с дискет)

Общая инструкция:
 - Установить СКАД "Сигнатура" 5.x (x86)
 - создать профиль СКАД "Сигнатура", установить сертификаты и пр.
     например, имя прояиля "IAS07"
 - с помощью rawwrite создать образ своей ключевой дискеты 
     имя файла должно быть: {имя_образа}.img (напр. "ias07.img")
   Полученный файл .img поместить в каталог пограммы
 - выполнить настройку программы.
   (конфиг хранится в файле cp365.ini в каталоге программы)
